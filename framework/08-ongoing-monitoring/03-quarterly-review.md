# 03 — Quarterly Review

## Sources

| Source | Reference |
|--------|-----------|
| ETA Framework | Point 7: Quarterly formal review points minimum |
| AI Playbook | Principle 5: Managing the full AI lifecycle |

## Purpose

The ETA Framework mandates formal review points at minimum quarterly. These reviews assess whether the LLM continues to meet all framework requirements and whether circumstances have changed.

## Quarterly Review Agenda

### 1. Performance review
- [ ] Review all KPIs against targets (see 08-monitoring/01-performance-monitoring)
- [ ] Identify trends (improving, stable, degrading)
- [ ] Action plan for any metrics below threshold

### 2. Fairness review
- [ ] Review bias monitoring results (see 08-monitoring/02-bias-drift-detection)
- [ ] Any new equality concerns?
- [ ] EIA still current?

### 3. Incident review
- [ ] Review any incidents since last quarter (see 08-monitoring/04-incident-response)
- [ ] Root causes identified and addressed?
- [ ] Systemic issues?

### 4. User feedback review
- [ ] Summary of user feedback and complaints
- [ ] Patterns or themes identified
- [ ] Actions taken in response

### 5. Compliance check
- [ ] ATRS record still accurate?
- [ ] DPIA still current?
- [ ] Any regulatory changes to address?
- [ ] Legal compliance still confirmed?

### 6. Technology review
- [ ] Model version changes?
- [ ] Supplier changes or issues?
- [ ] Security threats or vulnerabilities?
- [ ] Performance optimisation opportunities?

### 7. Governance review
- [ ] SRO and governance structure still appropriate?
- [ ] Team capability still sufficient?
- [ ] Budget and resources adequate?

### 8. Forward look
- [ ] Anticipated changes in next quarter
- [ ] Risks on the horizon
- [ ] Improvements planned

## Decisions

| Decision | Options |
|----------|---------|
| Continue as-is | No changes needed |
| Continue with improvements | Specific changes identified and planned |
| Pause for remediation | Significant issues require fixing before continuing |
| Decommission | System no longer meeting requirements (see 08-monitoring/05-end-of-life) |

## Output

Quarterly review report signed by SRO, with decisions and action items.
