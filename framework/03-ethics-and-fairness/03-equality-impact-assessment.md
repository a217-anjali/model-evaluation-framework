## Purpose

Conduct a formal Equality Impact Assessment before deploying the LLM. This is a legal requirement under the Public Sector Equality Duty — not optional guidance.

## EIA Process

### Step 1: Scope
- [ ] Define the LLM service and its intended users
- [ ] Identify which protected characteristics are most at risk
- [ ] Map the decision pathways where the LLM has influence

### Step 2: Evidence gathering
- [ ] Collect demographic data on intended user base
- [ ] Review bias audit results (see 03-ethics/02-bias-audit)
- [ ] Consult with representative groups and communities
- [ ] Review comparable services for known equality issues

### Step 3: Impact assessment

For each protected characteristic, assess:

| Age | | | | |

### Step 4: Mitigations
- [ ] For each negative impact, define specific mitigations
- [ ] Assess whether mitigations are sufficient
- [ ] If impacts cannot be mitigated, document justification or reconsider deployment

### Step 5: Monitoring
- [ ] Define ongoing monitoring for equality impacts
- [ ] Set review dates (link to Section 08 quarterly reviews)
- [ ] Establish feedback mechanisms for affected groups
