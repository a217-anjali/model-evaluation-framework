# Model Evaluation Framework for LLMs in the UK Public Sector

## Purpose

This framework provides a structured, evidence-based approach to evaluating Large Language Models (LLMs) for use in the UK public sector. Every requirement in this framework is traceable to one or more mandatory UK government standards, frameworks, or guidance documents.

## Scope

This framework applies to any UK public sector organisation considering, procuring, deploying, or operating an LLM-based system — whether built in-house, procured from a vendor, or accessed via API.

## How to Use This Framework

Follow the sections in order. Each section represents a stage in the LLM lifecycle:

| Section | Stage | Key Question |
|---------|-------|-------------|
| 01 Pre-Assessment | Discovery | Should we use an LLM at all? |
| 02 Legal & Regulatory | Compliance | Is it lawful? |
| 03 Ethics & Fairness | Ethics | Is it fair and responsible? |
| 04 Technical Evaluation | Testing | Does it actually work? |
| 05 Transparency & Accountability | Governance | Can we explain and own it? |
| 06 Procurement | Commercial | How do we buy it right? |
| 07 Deployment Gate | Go/No-Go | Are we ready to launch? |
| 08 Ongoing Monitoring | Live | Is it still working properly? |
| 09 Templates | Tools | What documents do we need? |
| 10 Reference | Sources | Where does this all come from? |
